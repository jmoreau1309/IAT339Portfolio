Jennifer Moreau

Project Goal

This project was assigned in my Graphic Design class as a group project. I worked with two classmates to create a magazine layout with a theme of our choosing. We each came up with theme ideas and draft layouts for those. We ended up choosing one of my teammates idea for a fashion magazine, but implemented some of the design decisions of each layout.

Design Decisions

I came up with the idea to use gold as a theme colour since it can symbolize luxury and importance which made sense for a modern fashion magazine. One of my teammates chose the font which is the font Vogue uses so it made our magazine look really on trend. The body text we wanted to be minimalist to remain easy to read and simple. We worked on the final product and presentation together, but I made sure to keep the layout rather simple as we were talking about modern fashion basics and I wanted to make sure the magazine had a clean cut look.

Wireframing

We used InDesign to create our gridlines so we could properly align items in the magazine and to make sure we have an appropriate sized rag. In addition to the web layout we made an interactive PDF version.

Summary

All in all, this project developed my team collaboration skills and helped tailor my design process. Working on lots of aligning, readjusting, and working with white space. I had never thought about these things all too much as they didn't seem that effective to me, but this project completely open my eyes.
